# 📞 Call Tracker Dashboard

Локальний дашборд для моніторингу ефективності дзвінків.
Зберігає дані в `calls.json` поруч з виконуваним файлом.

## Структура

```
calldash/
├── main.go       ← Go сервер (API + статика)
├── index.html    ← фронтенд
├── go.mod
└── calls.json    ← з'явиться автоматично після першого запуску
```

## Запуск

```bash
# 1. Зібрати
go build -o calldash .

# 2. Запустити
./calldash

# Windows:
go build -o calldash.exe .
calldash.exe
```

Відкрий у браузері: http://localhost:8080

## Як користуватись

| Кнопка | Що робить |
|--------|-----------|
| 📞 Дзвінок | +1 до загальних (дзвінок не взяли / завершився без результату) |
| ✅ Прийнято | +1 до загальних +1 до прийнятих |
| 🤝 Погоджено | +1 до загальних +1 до прийнятих +1 до погоджених |

Лічильники на кнопках показують **поточну годину**.
Графіки оновлюються одразу після натискання.

## API endpoints

```
POST /api/call?type=total      ← просто дзвінок
POST /api/call?type=accepted   ← прийнятий
POST /api/call?type=agreed     ← погоджений

GET  /api/today                ← слоти сьогодні (по годинах)
GET  /api/history              ← всі дні (підсумки)
GET  /api/now                  ← поточний час
```

## Формат calls.json

```json
{
  "days": [
    {
      "date": "2026-06-04",
      "slots": [
        { "hour": "09:00", "total": 5, "accepted": 3, "agreed": 1 },
        { "hour": "10:00", "total": 12, "accepted": 8, "agreed": 4 }
      ]
    }
  ]
}
```

Можна редагувати вручну — сервер читає файл при старті.
